\# 🏡 Real Estate Investment Analyzer

\## 📌 Project Overview This Streamlit app helps users: - Analyze
property investment quality - Predict future price - Explore real estate
trends

\## 🚀 Features - Investment Score Calculator - Future Price
Prediction - Market Insights Dashboard - Investment Simulator

\## 🛠 Tech Stack - Python - Streamlit - Pandas - Matplotlib
